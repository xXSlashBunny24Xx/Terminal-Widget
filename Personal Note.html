Bantuan Personal Note App

Selamat Datang!

Ini adalah aplikasi catatan pribadi Anda. Semua catatan disimpan di local cache browser Anda.

Manajemen Catatan

  • Judul Catatan (Atas): Judul catatan yang sedang aktif. Klik ikon pensil di sebelahnya untuk mengganti nama.
  • My Recent Note (di kanan): Dropdown untuk melihat dan beralih di antara semua catatan Anda. Anda juga dapat menghapus catatan dari daftar ini menggunakan ikon tempat sampah.
  • New Note (di bar atas): Membuat catatan baru yang kosong dan menyimpannya ke daftar Anda.
  • Clear: Menghapus seluruh isi catatan yang sedang Anda buka.

Penyimpanan & Ekspor/Impor

  • Penyimpanan Otomatis: Catatan Anda disimpan secara otomatis setiap kali Anda mengetik.
  • Export: Mengunduh semua catatan Anda sebagai satu file personal_notes.json.
  • Import: Membuka file .json untuk mengganti semua catatan Anda saat ini.

Fitur Editor

Gunakan tombol-tombol di controls bar untuk memformat teks Anda (Perataan, Ukuran Font, Bold, Italic, Underline, dan List).
