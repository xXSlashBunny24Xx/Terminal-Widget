Bantuan Take Note App

Selamat Datang!

Ini adalah editor catatan sederhana yang dibangun menggunakan gaya visual yang Anda berikan.

Fitur Editor

Gunakan tombol-tombol di controls bar (di bawah judul) untuk memformat teks Anda.

  • Dropdown (cth: Align Left): Untuk mengubah perataan teks (kiri, tengah, kanan, justify).
  • Dropdown (cth: Normal): Untuk mengubah ukuran font.
  • B, I, U: Untuk membuat teks menjadi Bold, Italic, atau Underline.
  • • List: Untuk membuat daftar poin (bullet list).

Penyimpanan & Ekspor/Impor

  • Penyimpanan Otomatis: Catatan Anda disimpan secara otomatis di browser (local cache) setiap kali Anda mengetik.
  • Export: Mengunduh catatan Anda saat ini sebagai file note.json.
  • Import: Membuka file .json dari komputer Anda untuk dimuat ke editor (akan menimpa catatan saat ini).
  • Clear: Menghapus semua konten di editor.

Area Penulisan

Area utama di bawah controls bar adalah tempat Anda menulis. Klik di sana dan mulailah mengetik. Semua catatan Anda bersifat sementara dan hanya ada di sesi browser ini.
